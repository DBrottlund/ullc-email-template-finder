Scans and discovers all email templates and their triggers in a WordPress/WooCommerce installation.

== Description ==

ULLC Email Template Finder helps WordPress administrators discover and manage all email templates in their installation by scanning plugin code, active theme, functions.php, and finding email-related code stored in the database.

= Features =

* Scans for email templates in plugins, themes, and database
* Tracks template locations and trigger points
* Provides a comprehensive list of all email functionality

= Translators =

* English (en_US) - default
* Translation ready - .pot file included

Want to contribute with a translation? Please contact us.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ullc-email-template-finder`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Email Templates screen to view discovered templates

== Frequently Asked Questions ==

= How do I contribute a translation? =

The plugin includes a .pot file in the /languages directory. You can use this file to create translations.
Please contact us if you'd like to contribute a translation.
